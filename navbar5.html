3D Navbar with Animated Flowchart Dropdown

New Feature: Hover over any navbar button to see the beautiful animated dropdown with flowchart lines!

Each dropdown is connected to its parent button with a glowing flowchart line that animates with a pulsing effect.

The dropdown items have smooth animations and color-coordinated hover effects matching their parent button.

This enhanced navbar maintains all the original 3D directional light effects while adding beautiful dropdown menus.

The flowchart lines create a visual connection between the main buttons and their dropdowns, enhancing the cyberpunk aesthetic.

Each dropdown item has its own subtle animations and glow effects that match the parent button's color scheme.
